◆ triangularFactorizationCrouts()

Matrix Matrix::triangularFactorizationCrouts ( )

Factorizes a given matrix into product of Lower and Upper triangular matrices.

The diagonal elements of the lower triangular matrix are non-zero. The diagonal elements of the upper triangular matrix are '1'.

Returns
An augmented matrix whose Upper triangular elements is the upper triangular matrix and the diagonals are '1' and Lower triangular elements is the lower triangular matrix including the diagonal elements.
